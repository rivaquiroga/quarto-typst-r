## Preparing for the workshop

You will need recents versions of both Quarto and R: 

- R >= 4.2
- [Quarto >= 1.6](https://quarto.org/docs/get-started/)

We will use the following packages: `quarto`, `knitr`, `tidyverse`, `gt`.
